## #문제

세 개의 장대가 있고 첫 번째 장대에는 반경이 서로 다른 n개의 원판이 쌓여 있다. 각 원판은 반경이 큰 순서대로 쌓여있다. 이제 수도승들이 다음 규칙에 따라 첫 번째 장대에서 세 번째 장대로 옮기려 한다.

1. 한 번에 한 개의 원판만을 다른 탑으로 옮길 수 있다.
2. 쌓아 놓은 원판은 항상 위의 것이 아래의 것보다 작아야 한다.

이 작업을 수행하는데 필요한 이동 순서를 출력하는 프로그램을 작성하라. 단, 이동 횟수는 최소가 되어야 한다.

<br/>

## #입력

첫째 줄에 첫 번째 장대에 쌓인 원판의 개수 N (1 ≤ N ≤ 20)이 주어진다.

<br/>

## #출력

첫째 줄에 옮긴 횟수 K를 출력한다.

두 번째 줄부터 수행 과정을 출력한다. 두 번째 줄부터 K개의 줄에 걸쳐 두 정수 A B를 빈칸을 사이에 두고 출력하는데, 이는 A번째 탑의 가장 위에 있는 원판을 B번째 탑의 가장 위로 옮긴다는 뜻이다.

<br/>

## #예제 입/출력

| 예제 입력 | 예제 출력 |
| ----- | ----- |
| 3 | 7<br/>1 3<br/>1 2<br/>3 2<br/>1 3<br/>2 1<br/>2 3<br/>1 3 |

## #풀이

또또또 하노이탑 문제; 수학공부할 때도 주구장창 나왔는데 여기서도 또 나온다ㅋㅋㅋㅋ

우선 하노이탑의 일반항은

타워 a, b, c로 원판 n개를 옮기는 경우의 수를 각각 A(n), B(n), C(n) 이라고 할 때

> C(n) = 2 * B(n-1) + 1

이 일반항을 풀면 최소로 이동하여 옮기는 횟수는 

> 2^n - 1

너무 유명하기에....외워두자

자 이제 이동하는 최소 횟수는 구했고, 과정을 구해야 한다.

알아야할 정보는 옮기는 원판의 개수, 있던 타워, 갈 타워, 나머지 타워 1개 이렇게 4가지가 필요하다.

그리고 조건들을 생각해보면

> 1. 옮기는 원판의 개수 = 1 이면 움직이는 경로를 출력할 것이다.
> 
> 2. 나머지 경우:
> &nbsp  - n-1개를 나머지 타워로 보낸다.
> &nbsp  - 맨 밑에 남아있는 원판 1개를 목표한 타워로 옮긴다.
> &nbsp  - 나머지 타워에 있는 n-1개의 원판을 목표한 타워로 옮긴다.

끝! 이제 코드를 작성해보자아아

```javascript
const process = [];


function moving(n, a, b, c) { //a: 있었던 곳, b: 갈 곳, c: 나머지 하나
    if (n == 1) {
        process.push(a + ' ' + b);
    }
    else {
        moving(n - 1, a, c, b);
        moving(1, a, b, c);
        moving(n - 1, c, b, a);
    }
}


function solution(data) {
    const n = parseInt(data);
    const result = Math.pow(2, n) - 1;
    console.log(result);
    moving(n, 1, 3, 2);

    console.log(process.join('\n'));
}


let fs = require('fs');
let input = fs.readFileSync('/dev/stdin').toString();

solution(input);
```

process 배열은 이동경로가 담길 배열이다.

moving 함수에서 앞서 설명했던 과정을 반복한다. 옮길 원판이 1개이면 process 배열에 이동 경로를 추가시키고, 그외의 경우에는 n-1개의 원판을 나머지 타워에 보내고, 맨 밑 1개를 목표 타워에 보내고, 나머지 타워에 있던 n-1개를 다시 목표한 타워로 옮기는 과정을 실행한다.

결국, 문제에 주어진 과정을 그대로 반복하면 되는 문제였다.
