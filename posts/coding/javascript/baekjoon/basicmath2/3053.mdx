## #문제

19세기 독일 수학자 헤르만 민코프스키는 비유클리드 기하학 중 택시 기하학을 고안했다.

택시 기하학에서 두 점 T1(x1,y1), T2(x2,y2) 사이의 거리는 다음과 같이 구할 수 있다.

D(T1,T2) = |x1-x2| + |y1-y2|

두 점 사이의 거리를 제외한 나머지 정의는 유클리드 기하학에서의 정의와 같다.

따라서 택시 기하학에서 원의 정의는 유클리드 기하학에서 원의 정의와 같다.

원: 평면 상의 어떤 점에서 거리가 일정한 점들의 집합

반지름 R이 주어졌을 때, 유클리드 기하학에서 원의 넓이와, 택시 기하학에서 원의 넓이를 구하는 프로그램을 작성하시오.

<br/>

## #입력

첫째 줄에 반지름 R이 주어진다. R은 10,000보다 작거나 같은 자연수이다.

<br/>

## #출력

첫째 줄에는 유클리드 기하학에서 반지름이 R인 원의 넓이를, 둘째 줄에는 택시 기하학에서 반지름이 R인 원의 넓이를 출력한다. 정답과의 오차는 0.0001까지 허용한다.

<br/>

## #예제 입/출력

| 예제 입력 | 예제 출력 |
| ----- | ----- |
| 1 | 3.141593<br/>2.000000 |
| 21 | 1385.442360<br/>882.000000 |
| 42 | 5541.769441<br/>3528.000000 |

<br/>

## #풀이

원의 넓이를 구하는 공식을 정리하면

> 유클리드 기하학
> : R^2 * PI

택시 기하학에서의 거리란, 각 좌표간의 거리에 해당한다. 즉, 택시 기하학에서 거리가 R인 점들은 마름모 형태가 나온다!

간단한 예를 들면, 거리 1인 점들의 집합은 (1, 0), (-1, 0), (0, 1), (0, -1) 을 연결했을 때 나오는 도형이고 이는 꼭짓점의 좌표가 R인 마름모이다.

즉, 넓이를 구하면

> 택시 기하학
> : 2 * R^2

끝!

```javascript
function solution(data) {
    const r = parseInt(data);
    const s1 = (r * r * Math.PI).toFixed(6);
    const s2 = 2 * r * r;

    console.log(s1 + '\n' + s2);
}

let input = require('fs').readFileSync('/dev/stdin').toString();

solution(input);
```
