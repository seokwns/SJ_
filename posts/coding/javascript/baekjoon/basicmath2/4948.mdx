## #문제

베르트랑 공준은 임의의 자연수 n에 대하여, n보다 크고, 2n보다 작거나 같은 소수는 적어도 하나 존재한다는 내용을 담고 있다.

이 명제는 조제프 베르트랑이 1845년에 추측했고, 파프누티 체비쇼프가 1850년에 증명했다.

예를 들어, 10보다 크고, 20보다 작거나 같은 소수는 4개가 있다. (11, 13, 17, 19) 또, 14보다 크고, 28보다 작거나 같은 소수는 3개가 있다. (17,19, 23)

자연수 n이 주어졌을 때, n보다 크고, 2n보다 작거나 같은 소수의 개수를 구하는 프로그램을 작성하시오. 

<br/>

## #입력

입력은 여러 개의 테스트 케이스로 이루어져 있다. 각 케이스는 n을 포함하는 한 줄로 이루어져 있다.

입력의 마지막에는 0이 주어진다.

<br/>

## #출력

각 테스트 케이스에 대해서, n보다 크고, 2n보다 작거나 같은 소수의 개수를 출력한다.

<br/>

## #예제 입/출력

| 예제 입력 | 예제 출력 |
| ----- | ----- |
| 1<br/>10<br/>13<br/>100<br/>1000<br/>10000<br/>100000<br/>0 | 1<br/>4<br/>3<br/>21<br/>135<br/>1033<br/>8392 |

<br/>

## #풀이

앞서 풀었던 문제들과 똑같다! 주어진 범위 안의 수들이 소수인지 판별만 한다면 단순 카운트 문제.

```javascript
const set = new Set();
const max = 123456 * 2;

for (let i = 3; i <= max; i += 2) {
    set.add(i);
}

set.add(2);

for (let i = 3; i < Math.sqrt(max) + 1; i += 2) {
    if (set.has(i)) {
        for (let k = i * i; k <= max; k += i) {
            set.delete(k);
        }
    }
}
```

자 소수 판별이 끝났다.

다음은? 판별한 결과들을 가지고 소수의 개수만 카운트하면 끝

```javascript
let result = '';
let index = 0;

while(data[index] !== '0') {
    let n = parseInt(data[index]);
    let count = 0;

    for (let i = n + 1; i <= 2 * n; i++) {
        if (set.has(i)) count++;
    }

    result += count + '\n';
    index++;
}


console.log(result);
```

data는 마찬가지로 문제에서 주어지는 입력값.

주어진 범위 안의 수들이 set에 존재하는지 아닌지, 존재하면 count++ 해주고 출력만 하면 끝
