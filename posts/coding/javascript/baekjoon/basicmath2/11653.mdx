## #문제

정수 N이 주어졌을 때, 소인수분해하는 프로그램을 작성하시오.

<br/>

## #입력

첫째 줄에 정수 N (1 ≤ N ≤ 10,000,000)이 주어진다.

<br/>

## #출력

N의 소인수분해 결과를 한 줄에 하나씩 오름차순으로 출력한다. N이 1인 경우 아무것도 출력하지 않는다.

<br/>

## #예제 입/출력

| 예제 입력 | 예제 출력 |
| ----- | ----- |
| 72 | 2<br/>2<br/>2<br/>3<br/>3 |
| 3 | 3 |
| 6 | 2<br/>3 |
| 2 | 2 |
| 9991 | 97<br/>103 |

<br/>

## #풀이

소인수 분해는? 2부터 소수인 숫자들을 더이상 나누어 떨어지지 않는 순간까지 나눈 결과라고 할 수 있다. 

기존에 에라토스테네스의 채로 소수를 판별할 때, 2부터 배수들을 지우는 과정을 했는데 이 과정들이 사실은 소인수 분해의 과정이였고, 이를 기록만 하면 끝인 문제이다.

```javascript
function solution(data) {
    let n = parseInt(data);

    if (n === 1) {
        return;
    }

    let result = [];

    for (let i = 2; i <= n; i++) {
        while (n%i === 0) {
            n /= i;
            result.push(i);
        }
    }


    console.log(result.join('\n'));
}
```

i가 2부터 1씩 증가시키는 for문을 사용하여 n을 i로 나누어 떨어질 때 까지 나누고, result 배열에 해당하는 i를 추가하면 끝! 소인수 분해를 하는 과정을 차례차례 구현하면 쉬운 문제같다.
