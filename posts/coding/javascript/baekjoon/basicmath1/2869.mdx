## #문제

땅 위에 달팽이가 있다. 이 달팽이는 높이가 V미터인 나무 막대를 올라갈 것이다.

달팽이는 낮에 A미터 올라갈 수 있다. 하지만, 밤에 잠을 자는 동안 B미터 미끄러진다. 또, 정상에 올라간 후에는 미끄러지지 않는다.

달팽이가 나무 막대를 모두 올라가려면, 며칠이 걸리는지 구하는 프로그램을 작성하시오.

<br/>
<br/>

## #입력

첫째 줄에 세 정수 A, B, V가 공백으로 구분되어서 주어진다. (1 ≤ B < A ≤ V ≤ 1,000,000,000)

<br/>
<br/>

## #출력

첫째 줄에 달팽이가 나무 막대를 모두 올라가는데 며칠이 걸리는지 출력한다.

<br/>
<br/>

## #예제 입/출력

- input: 2 1 5
    output: 4

- input: 5 1 6
    output: 2

- input: 100 99 1000000000
    output: 999999901

<br/>
<br/>

## #풀이

이건 조금 고민을 많이 했다,,,,낮과 밤 두가지 경우가 있는데 낮에 도달하면 더 이상 움직이지 않고 밤에 미끌어지지 않기 때문이다. 그래서 처음에는 낮/밤 순서대로 더하고 빼기를 계속하려고 했지만 숫자가 커질수록, 반복횟수가 많아질수록 너무 비효율적이다. 그래서 계산을 최소화하기 위해서 가장 먼저 든 생각은 하루 이동할 수 있는 거리를 가야할 거리로 나누면? 그 몫은 당연히 며칠 걸리는지 나올 것이다. 근데 여기서 문제, 하루 단위로 움직이는게 아니라 낮에 올라가고 밤에 미끌어지는 과정이다. 여기서 낮에 정상에 도달하게 된다면, 미끌어지지 않고 상황은 종료된다는 점이다. 

그래서 한 생각은 최종적으로 이동할 거리에서 낮에 이동하는 거리 a보다 적은 위치까지 가는데 걸리는 시간을 계산한다. 그리고 그 위치에서 a만큼 올라갔을 때, 정상에 도달하면? 앞서 계산한 날에서 1일만 더해주면 끝이다. 반대로 a만큼 올라갔는데 정상에 도달하지 못했다면, 최종적으로 이동할 거리를 하루동안 이동하는 거리로 나눈 값이 정답이 되는 것이다.

<br/>
이걸 코드로 옮기면 다음과 같다.

```javascript
function solution(a, b, v) {
    let day = 1;
    let one_day = a - b;

    let _day = Math.ceil((v - a) / one_day);

    if (_day * one_day + a >= v) {
        day = _day + 1;
    }
    else {
        day = Math.floor(v / one_day);
    }



    console.log(day);
}
```

여기서 주의해야 할 점은, 버릴지 아니면 올릴지 결정하는 점이다. v-a 까지 이동한 거리를 계산할 때에는 올려야 한다. 예를 들어 3.7일은 3일차에 올라갔다가 떨어지는 과정일 것이다. 다음날 움직임을 계산하기 위해서는 전날 모든 이동이 끝나야 하므로 '올림'을 해야 한다. 그 후 a를 더해서 정상 도달 여부를 계산해야 한다.
