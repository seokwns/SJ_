### 디자인 개편

몇번째 바꾸는 디자인인지....이것저것 해보다가 결국에는 처음 생각했던 디자인으로 돌아왔다 ㅋㅋㅋㅋ 왼쪽 썸네일, 오른쪽에 제목 + 미리보기 + 태그 + 날짜 이게 가장 깔끔한 것 같다. 

이제 여백 수치랑 자잘한 부분들만 더 손봐주면 얼추 완성각이 보임 ㅎㅎ 이젠 진짜 안바꿀거야.....여기서 좀 다듬고 디자인 공부해야지...

![포스트이미지](../../../post-images/20211029/포스트개편.png)

우선 구분선 좀 바꾼 후에 글자색이랑 높이 조절 좀 하면 나름 괜찮을듯? 아니면 내용 미리보기를 없애버릴까...먼가 조잡한 느낌 깔끔하게 태그 + 제목 + 날짜 이게 좋을 것 같기도 하고 흠....
